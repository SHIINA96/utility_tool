# JSON to CSV 转换工具

这是一个用于将 JSON 文件转换为 CSV 文件的 Python 工具，特别适用于处理包含中文内容的数据。

## 主要功能

1. 将 JSON 数据转换为 CSV 格式
2. 自动处理中文编码，确保在 Excel 中正确显示
3. 自动添加序号列（从1开始）
4. 智能处理换行符（将文本中的换行符替换为空格）
5. 支持处理单个对象或对象数组的 JSON 数据

## 使用方法

1. 准备输入文件：
   - 将您的 JSON 文件命名为 `input.json`
   - 放在与脚本相同的目录下

2. 运行脚本：
   ```bash
   python json2csv_simplify_ch.py
   ```

3. 查看结果：
   - 转换后的文件将保存为 `output.csv`
   - 可以直接用 Excel 打开，中文内容将正确显示

## 输入文件格式示例

```json
[
    {
        "姓名": "张三",
        "年龄": 25,
        "城市": "北京",
        "描述": "这是一段\n包含换行符的\n文本"
    },
    {
        "姓名": "李四",
        "年龄": 30,
        "城市": "上海",
        "描述": "普通文本"
    }
]
```

## 输出文件特点

1. 第一列为自动生成的序号（从1开始）
2. 使用 UTF-8-BOM 编码，确保 Excel 正确显示中文
3. 文本中的换行符会被替换为空格
4. 自动识别所有字段名

## 注意事项

- 输入的 JSON 文件必须使用 UTF-8 编码
- 如果需要处理其他文件名，可以修改脚本中的 `json_file_path` 和 `csv_file_path` 变量
- 该工具会自动处理 JSON 中的所有字段，即使不同对象包含不同的字段

## 环境要求

- Python 3.x
- 无需安装额外的依赖包，使用 Python 标准库
